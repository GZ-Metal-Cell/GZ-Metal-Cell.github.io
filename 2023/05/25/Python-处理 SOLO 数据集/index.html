    • BBoxSizeStatsAnalyzer зұ»з”ЁдәҺеҲҶжһҗжЈҖжөӢз»“жһңдёӯ bounding box зҡ„еӨ§е°ҸеҲҶеёғжғ…еҶөгҖӮе®ғжІЎжңүжҺҘеҸ—еҸӮж•°пјҢеҸҜд»Ҙз”ЁдәҺд»»дҪ•еҢ…еҗ« bounding box дҝЎжҒҜзҡ„ж•°жҚ®йӣҶгҖӮйҖҡиҝҮ bbox_size_analyzer еҜ№иұЎпјҢеҸҜд»ҘиҺ·еҫ—жЈҖжөӢз»“жһңдёӯ bounding box еӨ§е°Ҹзҡ„з»ҹи®ЎдҝЎжҒҜпјҢдҫӢеҰӮжңҖе°ҸеҖјгҖҒжңҖеӨ§еҖјгҖҒе№іеқҮеҖје’Ңж–№е·®зӯүгҖӮ
    • BBoxHeatMapStatsAnalyzer зұ»з”ЁдәҺз”ҹжҲҗй’ҲеҜ№жЈҖжөӢз»“жһңдёӯ bounding box зҡ„зғӯеӣҫгҖӮе®ғд№ҹжІЎжңүжҺҘеҸ—еҸӮж•°пјҢеҸҜд»Ҙз”ЁдәҺд»»дҪ•еҢ…еҗ« bounding box дҝЎжҒҜзҡ„ж•°жҚ®йӣҶгҖӮйҖҡиҝҮ bbox_hmap_analyzer еҜ№иұЎпјҢеҸҜд»Ҙе°ҶжЈҖжөӢз»“жһңдёӯ bounding box зҡ„дҪҚзҪ®дҝЎжҒҜжҳ е°„еҲ°дёҖеј еӣҫзүҮдёҠпјҢе№¶дё”еҸҜи§ҶеҢ–еҮә bounding box зҡ„еҜҶеәҰеҲҶеёғжғ…еҶөгҖӮ
    • StatsHandler зұ»еҸҜд»ҘжҺҘеҸ—дёҖдёӘеҢ…еҗ«з»ҹи®ЎдҝЎжҒҜзҡ„ Solo еҜ№иұЎдҪңдёәеҸӮж•°пјҢе№¶жҸҗдҫӣдәҶ handle ж–№жі•з”ЁдәҺеӨ„зҗҶеҲҶжһҗеҷЁпјҲAnalyzerпјүеҲ—иЎЁгҖӮеңЁжң¬дҫӢдёӯпјҢStatsHandler еҜ№иұЎзҡ„еҸӮж•°жҳҜ soloпјҢеҚізӣ®ж ҮжЈҖжөӢз»“жһңж•°жҚ®йӣҶгҖӮеҸҰеӨ–пјҢhandle ж–№жі•жҺҘеҸ—дёҖдёӘ Analyzer еҜ№иұЎеҲ—иЎЁдҪңдёәеҸӮж•°пјҢеҲҶеҲ«жҳҜ bbox_count_analyzerгҖҒbbox_hmap_analyzer е’Ң bbox_size_analyzerгҖӮ
    • handle ж–№жі•иҝ”еӣһдёҖдёӘз»“жһңеҜ№иұЎпјҢе…¶дёӯеҢ…еҗ«дәҶд»ҘдёҖе®ҡеҪўејҸеӮЁеӯҳзҡ„дёүз§ҚеҲҶжһҗеҷЁзҡ„з»ҹи®ЎдҝЎжҒҜж•°жҚ®гҖӮдҫӢеҰӮпјҢbbox_count_analyzer еҸҜд»Ҙиҫ“еҮә bounding box ж•°йҮҸзҡ„жңҖеӨ§еҖјгҖҒжңҖе°ҸеҖје’Ңе№іеқҮеҖјпјҢbbox_hmap_analyzer еҸҜд»Ҙиҫ“еҮә bounding box еҜҶеәҰзғӯеҠӣеӣҫж•°жҚ®пјҢbbox_size_analyzer еҸҜд»Ҙиҫ“еҮә bounding box еӨ§е°Ҹзҡ„з»ҹи®ЎдҝЎжҒҜгҖӮresult е°ұжҳҜиҝҷдәӣз»ҹи®ЎдҝЎжҒҜзҡ„йӣҶеҗҲгҖӮ
