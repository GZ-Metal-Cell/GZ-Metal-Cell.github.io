论文

检索

  • The latest in Machine Learning | Papers With Code

    • 最喜欢的论文检索网站，首页就有最新的论文，很多论文都附有 GitHub 代码仓库

  • arXiv.org e-Print archive

    • 可以找到很多还没没被期刊录用的论文，而且可以很方便地下载

  • dblp (uni-trier.de)

    • 可以找到几乎所有期刊的目录

  • Sci-Hub: 将知识带给每个人

    • 听师兄说可以输入想要的文献的题目或者 DOI 来获得论文，但是好像比较新的文献查找不到

  • MDPI SciProfiles

    • 这个网站可以把老师的名字输进去，看看师兄们都发了什么乱七八糟的论文，Electronics yyds！

  • 百度学术 - 保持学习的态度 (baidu.com)

    • 没怎么试过，不过可以看到很多老师很久以前发的中文文献

  • Google Scholar

    • 这也没怎么试过……

  • 浙江图书馆 (zjlib.cn)

    • 可以进知网下一些中文论文，不知道现在好不好使

  • 中国计算机学会推荐国际学术会议和期刊目录-2022

写作

  • 不得不收藏的深度神经网络结构图绘制方法 - 知乎 (zhihu.com)

    • 线下用 $\LaTeX$ 确实麻烦了点，还是线上方便些
  • LaTeX 工作室 (latexstudio.net)
    • 找个模板啥的

科研

课程

  • ML 2023 Spring (ntu.edu.tw)

    • 李宏毅老师的深度学习课程我最满意，中文教学，课堂也挺活泼，吴恩达讲英文给我听困了，我菜😅

  • 课程安排 - 动手学深度学习课程 (d2l.ai)
