【01-新手村】- 从Pr的最简工作流开始讲起

    导入素材，拖到时间轴中，就会创建一个序列。

    如果新拖放进来的视频大小不一致，可以右键 缩放为帧大小。

    一个序列包含三个视频序列和三个音频序列。

    C 对序列进行切割。

    V 对视频进行选择。

    可以使用 Ctrl + K 在当前时刻立即进行一次切割。

本章小结
  • PR 的用途
    PR 是一款用于视频段落的组合和拼接，并提供一定的特效与调色功能。
  • 项目窗口
    用于存储视频，音频，图片，序列等素材。
  • 序列窗口
    用于编辑音频、视频，可以过个轨道同时编辑。
  • 预览窗口
    用于实时预览时间线上的素材效果。
  • 效果控件窗口
    可以对素材的基本属性做出调整，i也可以调整添加在素材上的效果参数。
  • 效果窗口
    PR 中需要用到的各种自带的效果，放在该窗口中。
  • PR 工作流程
    1. 素材窗口中导入素材
    2. 新建序列
    3. 通过剃刀工具 (C) 对素材进行裁剪
    4. 通过选择工具 (V) 对素进行选择并删除
    5. 通过拼接完成对素材的剪辑
    6. 渲染输出成片

【02-新手村】- 剪辑多素材如何标记出入点

    单独选择一个源，按 I 设置入点，O 设置出点。

    时间轴中，M 标记时间点，Shift + M 转到下一个标记的时间点。

    右键 剪辑速度/持续时间 改变视频片段的播放的速度。

    有些视频会自带音轨，拖动到视频轨道会覆盖对应的音轨。右键 取消链接 再把音轨删了。

    最后一阵操作。

本章小结
  • 导入素材
    • 熟悉素材，选取合适的音乐。
    • 将素材和音乐导入到 pr 素材窗口中。
  • 剪辑流程
    • 反复感受音乐，对音乐做一个节奏和情绪上的区分。
    • 选取偏努力和刻苦一类的素材，在素材窗口中，选择出点和入点，截取合适的段落，讲素材拖到视频轨道上。
    • 对于前半段音乐节奏比较快的地方，选取的素材相应的要短一些，剪辑点可以密一些。
    • 选取成功，胜利一类的素材，放在音乐后半段。
    • 反复感受音乐节奏，可以在音乐有大的节拍的地方，将剪辑点对齐。
    • 反复预览视频，使视频剪辑点和音乐更加契合。
    • 导出视频。

【03-新手村】- 如何制作抽帧/卡点视频

    导入音频，如果它声音聒噪，右键 音频增益 -20dB。

    观察音频中波形变化较大的时刻，抽帧（删掉视频中间一小段）。

    按 M 标记好波形较大的时刻。Shift + M 切换到下一个标记好的时刻。

【06-实战篇】- 推入拉出式转场制作

    导入素材，在需要转场的地方作一个调整图层。

    选择两个视频切换的那个时刻，Shift + ← / → 可以向前 / 后移动 5 帧。

    在调整图层里建一个 变换，缩放设为 50%。

    做四个 镜像。

    这几个步骤的目的是通过镜像扩充图像范围，模仿视野变广的效果。

    参数的设置应合理，使得镜像补全的画面不至于太突兀（如裙子上长腿，就不可取）。

    最后再上一个 变换，缩放 参数使用帧动画，值从 200 - 150。

    200 的取值是因为之前有一个 50 的缩放，50% x 200% = 100%，使得转场之前的画面与开始转场的画面连贯。

    150 的取值可以使得画面缩小，视野变广。

    调整 缩放 的速度曲线。

    增加效果 Alpha 调整，选择 忽略 Alpha。把 快门角度 的值调大，这样画面就有动态模糊的效果。

    将这些效果复制到出场的调整图层中。

    修改 变换 帧动画中 缩放 的值，值从 270 - 200。

    270 的取值可以使得画面放大，视野变窄。

    200 的取值是因为之前有一个 50 的缩放，50% x 200% = 100%，使得转场之后的画面与结束转场的画面连贯。

【07-实战篇】-制作酷炫三分分屏转场

    三分分屏处放三个等长的视频片段。

    对于每个视频片段，添加效果 线性擦除，擦除角度 设为 120°，过渡完成 即为擦除掉画面的百分比。

    调整这些参数以及位置，使得三分分屏画面协调。

    对于每个分屏，制作关键帧动画。

    使用 连续贝塞尔曲线 使得分屏切换速度协调。

    背景声太大，调小。
