    5 不一定有效率 GIL

    • 在处理较简单的问题时, 由于切换线程需要耗费时间(GIL), 多线程不一定比单线程用时更短(每个时刻只能有一个线程在CPU工作)
    • 为什么多线程有时候会比单线程耗时更短? 因为有些时候会边运算边读写数据. 解决方法: 多核运算
    • 分别用常规方法和多线程方法计算0-99999求和的四倍
