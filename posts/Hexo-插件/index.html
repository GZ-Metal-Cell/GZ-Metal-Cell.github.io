    事件

    • 事件 | Hexo

    Note

    ​ Hexo 继承了 EventEmitter。 您可以用 on 方法监听 Hexo 所发布的事件，也可以使用 emit 方法对 Hexo 发布事件。 更详细的说明请参阅 Node.js 的 API。

    ​ 感觉其它的功能更依赖于过滤器……

    执行命令 依次触发事件
    hexo cl ready-exit
    hexo new ready-new-exit
    hexo g & hexo s ready-generateBefore-generateAfter-exit
    hexo d ready-deployBefore-deployAfter-exit

    ready

    Note

    ​ 在初始化完成后发布。

    ​ Hexo 启动时会触发。

    new

    Note

    ​ 在文章文件建立后发布。 该事件返回文章参数。

    ​ 当执行命令 hexo new 时会触发：

    exit

    Note

    ​ 在 Hexo 结束前发布。
