OCPP 1.6J 介绍

OCPP（Open Charge Point Protocol） 是一种用于 电动汽车充电桩（Charge Point） 与 充电管理系统（Central System / CSMS） 之间通信的开放协议。

它定义了消息格式、通信流程、命令集，使不同厂家的设备能够互联互通。

典型的充电流程如下：

BootNotification 注册 → StatusNotification 报告可用 → Authorize 验证用户 → StartTransaction 开始充电 → MeterValues 上报数据 → StopTransaction 结束充电
