7. gRPC

gRPC 基于 HTTP/2 + Protobuf，典型用于微服务 / 后端对后端。浏览器原生不支持你写的这种 gRPC，需要 gRPC-Web 或网关。

本示例：

  1. 在项目根目录生成代码（只需一次）：
    python -m grpc_tools.protoc -I07_grpc --python_out=07_grpc --grpc_python_out=07_grpc 07_grpc/demo.proto
  2. 启动 gRPC 服务：
    python 07_grpc/grpc_server.py
  3. 另开终端调用：
    python 07_grpc/grpc_client.py
