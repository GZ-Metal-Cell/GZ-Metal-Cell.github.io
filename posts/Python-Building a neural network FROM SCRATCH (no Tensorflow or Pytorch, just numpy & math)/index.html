​ Our NN will have a simple two-layer architecture. Input layer a[0]a^{[0]}a[0] will have 784784784 units corresponding to the 784784784 pixels in each 28×2828\times 2828×28 input image. A hidden layer a[1]a^{[1]}a[1] will have 101010 units with ReLU activation, and finally our output layer a[2]a^{[2]}a[2] will have 101010 units corresponding to the ten digit classes with softmax activation.

​ 我们的神经网络将有一个简单的两层结构。输入层 a[0]a^{[0]}a[0] 将有 784784784 个单元，对应于每个 28×2828\times 2828×28 输入图像中的 784784784 个像素。隐藏层 a[1]a^{[1]}a[1] 将有 101010 个单元，用 ReLU 激活，最后我们的输出层 a[2]a^{[2]}a[2] 将有 101010 个单元，对应于用 softmax 激活的 101010 个数字类别。

Softmax

  • 一文详解Softmax函数

Softmax 函数将各个输出节点的输出值范围映射到 [0, 1]，并且约束各个输出节点的输出值的和为 1。

Softmax(zi)=eiz∑C=1CezCSoftmax(z_i)=\frac{e^z_i}{\sum^C_{C=1}e^{z_C}}Softmax(zi​)=∑C=1C​ezC​eiz​​，其中 ziz_izi​ 为第 iii 个节点的输出值，CCC 为输出结点的个数，即分类的类别个数。通过 Softmax 函数就可以将多分类的输出值转换为范围在 [0, 1] 和为 1 的概率分布。
