    How do we know when the ray intersects the box?

    вҖӢ жҲ‘д»¬еҰӮдҪ•зҹҘйҒ“е°„зәҝдҪ•ж—¶дёҺзӣ’еӯҗзӣёдәӨпјҹ

    • Key ideas

      е…ій”®жҖқжғі

      • The ray enters the box only when it enters all pairs of slabs

        еҸӘжңүеҪ“е°„зәҝиҝӣе…ҘжүҖжңүе№іжқҝеҜ№ж—¶пјҢе®ғжүҚдјҡиҝӣе…Ҙзӣ’еӯҗ

      • The ray exits the box as long as it exits any pair of slabs

        еҸӘиҰҒе°„зәҝзҰ»ејҖд»»дҪ•дёҖеҜ№е№іжқҝпјҢе®ғе°ұдјҡзҰ»ејҖзӣ’еӯҗ

    • For each pair, calculate the $t_{\text{min}}$ and $t_\text{max}$ (negative is fine)

      еҜ№дәҺжҜҸдёҖеҜ№пјҢи®Ўз®— $t_{\text{min}}$ е’Ң $t_\text{max}$пјҲиҙҹж•°д№ҹеҸҜд»Ҙпјү

    • For the 3D box, $t_{\text{enter}} = \max{t_{\text{min}}}, t_{\text{exit}} = \min{t_{\text{max}}}$

      еҜ№дәҺ 3D зӣ’еӯҗпјҢ$t_{\text{enter}} = \max{t_{\text{min}}}пјҢt_{\text{exit}} = \min{t_{\text{max}}}$

    • If $t_{\text{enter}} < t_{\text{exit}}$, we know the ray stays a while in the box
      (so they must intersect!)

      еҰӮжһң $t_{\text{enter}} < t_{\text{exit}}$пјҢжҲ‘д»¬зҹҘйҒ“е°„зәҝдјҡеңЁзӣ’еӯҗдёӯеҒңз•ҷдёҖж®өж—¶й—ҙ пјҲжүҖд»Ҙе®ғд»¬дёҖе®ҡзӣёдәӨпјҒпјү

    • However, ray is not a line

      дҪҶжҳҜпјҢе°„зәҝдёҚжҳҜзӣҙзәҝ

      • Should check whether $t$ is negative for physical correctness!
      • еә”жЈҖжҹҘ $t$ жҳҜеҗҰдёәиҙҹж•°д»ҘзЎ®дҝқзү©зҗҶжӯЈзЎ®жҖ§пјҒ

    • What if $t_\text{exit} < 0$?

      • The box is "behind" the ray вҖ” no intersection!

        зӣ’еӯҗеңЁе°„зәҝвҖңеҗҺйқўвҖқ вҖ”вҖ”жІЎжңүзӣёдәӨпјҒ

    • What if $t_{\text{exit}} \ge 0$ and $t_{text{enter}} < 0$?

      • The ray' s origin is inside the box вҖ” have intersection!

        зәҝзҡ„еҺҹзӮ№еңЁзӣ’еӯҗеҶ…йғЁ вҖ” жңүзӣёдәӨпјҒ

    Why: we want to solve an integral, but it can be too difficult to solve analytically.

    вҖӢ еҺҹеӣ пјҡжҲ‘д»¬жғіи§ЈдёҖдёӘз§ҜеҲҶпјҢдҪҶз”ЁеҲҶжһҗж–№жі•пјҲзүӣйЎҝиҺұеёғе°јиҢЁе…¬ејҸпјүи§ЈеӨӘйҡҫдәҶгҖӮ

    What & How: estimate the integral of a function by averaging random samples of the function' s value.

    вҖӢ еҶ…е®№е’Ңж–№жі•пјҡйҖҡиҝҮеҜ№еҮҪж•°еҖјзҡ„йҡҸжңәж ·жң¬жұӮе№іеқҮеҖјжқҘдј°и®ЎеҮҪж•°зҡ„з§ҜеҲҶгҖӮ

    Let us define the Monte Carlo estimator for the definite integral of given function $f(x)$

    вҖӢ и®©жҲ‘д»¬дёәз»ҷе®ҡеҮҪж•° $f(x)$ зҡ„е®ҡз§ҜеҲҶе®ҡд№үи’ҷзү№еҚЎжҙӣдј°и®ЎйҮҸ

    Definite integral е®ҡз§ҜеҲҶпјҡ$\int_a^bf(x)dx$

    Random variable йҡҸжңәеҸҳйҮҸпјҡ$X_i\sim p(x)$

    Monte Carlo estimator и’ҷзү№еҚЎжҙӣдј°и®ЎйҮҸпјҡ$F_N=\frac{1}{N}\sum_{i=1}^N\frac{f(X_i)}{p(X_i)}$

    Example: Uniform Monte Carlo Estimator

    вҖӢ зӨәдҫӢпјҡз»ҹдёҖи’ҷзү№еҚЎзҪ—дј°и®ЎйҮҸ

    Uniform random variable еҜ№дәҺеқҮеҢҖйҡҸжңәеҸҳйҮҸпјҡ

    $\begin{aligned}&X_i\sim p(x)=C \\&\begin{aligned}\int_a^bp(x)dx=1\end{aligned} \\&\Rightarrow \begin{aligned}\int_a^bCdx=1\end{aligned} \\&\Rightarrow \begin{aligned}C=\frac{1}{b-a}\end{aligned} \end{aligned}$

    Uniform random variable еқҮеҢҖйҡҸжңәеҸҳйҮҸпјҡ$X_i\sim p(x)=\frac1{b-a}$

    Basic Monte Carlo estimator еҹәжң¬и’ҷзү№еҚЎзҪ—дј°и®ЎйҮҸпјҡ$F_N=\frac{b-a}N\sum_{i=1}^Nf(X_i)$

    Solution: Russian Roulette (RR)

    вҖӢ и§ЈеҶіж–№жЎҲпјҡдҝ„зҪ—ж–ҜиҪ®зӣҳиөҢпјҲRRпјү

    Russian Roulette is all about probability

    вҖӢ дҝ„зҪ—ж–ҜиҪ®зӣҳиөҢе®Ңе…ЁжҳҜжҰӮзҺҮжёёжҲҸ

    вҖӢ жҰӮзҺҮ $0 < P < 1$пјҢеҲҷжІЎй—®йўҳ

    With probability $1 - P$, otherwise

    вҖӢ жҰӮзҺҮ $1 - P$пјҢеҸҰеӨ–жғ…еҶө

    Previously, we always shoot a ray at a shading point and get the shading result $L_o$

    вҖӢ д»ҘеүҚпјҢжҲ‘д»¬жҖ»жҳҜеҗ‘зқҖиүІзӮ№еҸ‘е°„е°„зәҝе№¶еҫ—еҲ°зқҖиүІз»“жһң $L_o$

    Suppose we manually set a probability $P (0 < P < 1)$

    вҖӢ еҒҮи®ҫжҲ‘д»¬жүӢеҠЁи®ҫзҪ®дёҖдёӘжҰӮзҺҮ $P (0 < P < 1)$

    With probability $P$, shoot a ray and return the shading result divided by $P$: $L_o / P$

    вҖӢ д»ҘжҰӮзҺҮ $P$пјҢеҸ‘е°„дёҖжқЎе°„зәҝе№¶иҝ”еӣһйҷӨд»Ҙ $P$ зҡ„зқҖиүІз»“жһңпјҡ$L_o / P$

    With probability $1-P$, don' t shoot a ray and you' ll get $0$

    вҖӢ д»ҘжҰӮзҺҮ $1-P$пјҢдёҚеҸ‘е°„е°„зәҝпјҢжӮЁе°Ҷеҫ—еҲ° $0$

    In this way, you can still expect to get $L_o$! :

    вҖӢ иҝҷж ·пјҢжӮЁд»Қз„¶еҸҜд»Ҙжңҹжңӣеҫ—еҲ° $L_o$пјҒпјҡ

    $E = P \cdot (L_o / P) + (1 - P) \cdot 0 = L_o$
